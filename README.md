# kushalpitaliya.github.io

Documentation and landing page for the [text-parsematch](https://github.com/KushalPitaliya/text-parsematch) Python package.

## About

This repository hosts the GitHub Pages site for **text-parsematch** -- a Python package for processing text inputs with pattern matching and structured output validation.

Live site: [kushalpitaliya.github.io](https://kushalpitaliya.github.io)

## Local Development

This site is built with Jekyll and hosted on GitHub Pages.

To run locally:

```
bundle install
bundle exec jekyll serve
```

Then visit http://localhost:4000.
